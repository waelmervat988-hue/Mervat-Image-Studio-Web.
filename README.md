موقع مرڤت لمعالجة الصور | Mervat Image Studio
​مشروع ويب متكامل لمعالجة الصور رقمياً باستخدام لغة Python، تم تطويره بأسلوب احترافي لتقديم تجربة مستخدم سلسة وإبداعية.
​📝 نبذة عن المشروع
​يعتبر هذا المشروع تطبيق ويب تفاعلي يعتمد على إطار العمل Flask، يتيح للمستخدمين رفع صورهم وتطبيق مجموعة من فلاتر المعالجة الرقمية المتقدمة باستخدام مكتبة OpenCV، مع إمكانية معاينة النتائج وتحميلها فوراً.
​✨ المميزات التقنية (Features)
​يحتوي الموقع على أدوات معالجة متنوعة تشمل:
​🎨 الفلاتر الكلاسيكية: تحويل الصور إلى الأبيض والأسود (Grayscale) وتأثير السيبيا (Sepia) الدافئ.
​🛠️ الأدوات الذكية:
​Canny Edge Detection: كشف حواف العناصر داخل الصورة بدقة.
​Invert Colors: عكس ألوان الصورة (Negative).
​Brightness Control: زيادة مستوى الإضاءة لتحسين وضوح الصور الباهتة.
​🤝 دمج الصور (Image Blending): تقنية دمج صورتين معاً بنسب متساوية لإنشاء تأثيرات فنية.
​📥 التحميل المباشر: زر مخصص لحفظ الصورة المعالجة على جهاز المستخدم بلمسة واحدة.
​🛠️ الأدوات والتقنيات المستخدمة (Tech Stack)
​تم بناء المشروع باستخدام أفضل التقنيات
  اللغة الأساسية: Python
​إطار عمل الويب: Flask
​معالجة الصور: OpenCV
